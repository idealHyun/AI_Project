# 실감인공지능 프로젝트

### 프로젝트 소개

위 프로젝트는 OpenAI의 GPT API를 활용하여 제품 설명을 통해 주요 특징을 추출하고,  
특징을 통해 리뷰 질문지를 생성하고, 해당 답변을 통해 하나의 품질있는 리뷰를 생성하고 해당 리뷰가 마케팅적으로 사용되는 것을 목표로합니다.

사이트 주소 : http://15.165.197.11:3000

### 주요 기능

API URL = http://15.165.197.11:3000

- 기능을 사용하기 전에는 API KEY를 발급하여 API 요청시 header에 API KEY를 넣어주어야합니다.
```javascript
       await axios.post(
        '/api_url',
        { body },
        {
          headers: {
            'api_key': API_KEY,
          },
        }
      );
```

### 제품 특징 추출
  - '/api/openai/features'
  - 제품 설명을 분석하여 전문 용어를 배제한 간단하고 이해하기 쉬운 표현으로 주요 특징을 추출합니다.

##### 요청

헤더:
| 키        | 값    | 설명      |
|:-:|:-:|:-:|
| `api_key` | `발급받은 API_KEY` | 인증을 위한 API 키 |

#### 바디 (JSON 형식):
| 필드  | 타입   | 설명     |
|:-:|:-:|:-:|
| `desc` | string | 분석할 제품 설명 |

**예시**:
```json
{
    "desc": "혁신적이고 가벼우며 휴대성이 뛰어난 진공 청소기입니다."
}
```
---

##### 응답

###### 상태 코드: 200 OK

###### 바디 (JSON 형식):
| 필드    | 타입   | 설명       |
|:-:|:-:|:-:|
| `category` | string | 제품 설명을 바탕으로 분석된 제품 카테고리 |
| `features` | array  | 제품과 관련된 주요 특징 리스트           |

**예시**:
```json
{
    "category": "가전제품",
    "features": ["가벼움", "휴대성", "혁신적"]
}
```

### 질문지 생성
  -  '/api/openai/questions'
  - 특징을 바탕으로 사용자가 대답하기 쉬운 질문을 자동으로 생성합니다.

##### 요청

헤더:
| 키        | 값     | 설명   |
|:-:|:-:|:-:|
| `api_key` | `발급받은 API_KEY`  | 인증을 위한 API 키 |

바디 (JSON 형식):
| 필드     | 타입   | 설명   |
|:-:|:-:|:-:|
| `category`  | string | 제품의 카테고리               |
| `features`  | string | 제품과 관련된 주요 특징 리스트 |

**예시**:
```json
{
    "category": "가전제품",
    "features": ["가벼움", "휴대성", "혁신적"]
}
```
##### 응답

###### 상태 코드: 200 OK
바디 (JSON 형식):
|필드	|타입	|설명|
|:-:|:-:|:-:|
|questions	|array	|리뷰 작성에 필요한 질문 리스트|
예시:

```json
{
    "questions": [
        "이 제품의 가장 마음에 드는 점은 무엇인가요?",
        "사용 중 개선이 필요하다고 느낀 점은 무엇인가요?",
        "제품의 휴대성이 만족스러웠나요?"
    ]
}
```

### 리뷰 생성
  - '/api/openai/answer'
  - 사용자가 작성한 질문에 대한 답변을 제출하여 리뷰 결과를 생성합니다.

##### 요청

헤더:
| 키      | 값        | 설명     |
|:-:|:-:|:-:|
| `api_key` | `발급받은 API_KEY`                | 인증을 위한 API 키 |

바디 (JSON 형식):
| 필드     | 타입   | 설명     |
|:-:|:-:|:-:|
| `answers` | array  | 질문과 답변 객체 리스트 (질문/답변 쌍) |

**예시**:
```json
{
    "answers": [
        {
            "question": "이 제품의 가장 마음에 드는 점은 무엇인가요?",
            "answer": "디자인이 세련되고 사용하기 편리합니다."
        },
        {
            "question": "사용 중 개선이 필요하다고 느낀 점은 무엇인가요?",
            "answer": "배터리 지속 시간이 조금 짧은 것 같습니다."
        }
    ]
}
```

##### 응답

###### 상태 코드: 200 OK

##### 바디 (JSON 형식):
| 필드      | 타입   | 설명                        |
|:-:|:-:|:-:|
| `answer`  | string | 제출된 답변을 기반으로 생성된 리뷰 결과 |

**예시**:
```json
{
    "answer": "이 제품은 세련된 디자인과 편리한 사용성을 제공하지만, 배터리 지속 시간이 아쉽습니다."
}
```
